MY WORK ABOUT ME

Engineering Specialities

Mechanical Design

I take parts from sketch to working assemblies in Fusion 360. I split designs so they print cleanly, add brass inserts for strong threads, and build in the right clearances so things fit first time. I choose materials to match the job (PLA/PETG for enclosures, ASA for sun, CF for loaded parts) and plan fasteners and sealing up front.

Electronic Engineering

I build small embedded systems end-to-end. I choose a microcontroller that fits the job, wire sensors and actuators, and bring the board up with a multimeter/logic analyzer. When stripboard isn't enough, I design simple PCBs with JLCPCB, plan power properly (battery, charging, protection). Firmware is clear and serviceable: talk to devices over common serial buses (I²C, SPI, UART, CAN), add status LEDs/logs, and fail safely. Recent work includes handling acoustic modem commands and driving a switchable magnet release. The goal is hardware you can trust, not a one-off bench demo.

Robotics Software

I write the software that ties the robot together. In Python and C++ using ROS 2, I implement control loops, device interfaces, and the small tools that make testing faster. I log and visualise data so faults are obvious, add simple health checks and timeouts, and package things so they run on a laptop or a small onboard computer without fuss. For vision tasks I use OpenCV; if ML adds value, I keep it lightweight and explainable. The aim is code that deploys cleanly and can be debugged on a bad day.

Engineering experience

Led embedded systems design for acoustic communication in the Ropeless Fishing Project, creating a custom PCB and firmware for RS-232 nanomodem integration.

Awarded the AWE Prize for best industrial placement and earned First-Class Honours in MEng Robotics, Autonomous and Interactive Systems at Heriot-Watt University.
